1.Nとして/しては/しても/してのN
Với tư cách/vị trí là/ Với tư cách/vị trí là...thì/ Cho dù là với tư cách/vị trí là...

Dùng trong trường hợp nói đến một sự vật, con người là đại biểu hoặc là một cá thể thuộc một nhóm chung nào đó

Ví dụ
彼( かれ ) は外国人( がいこくじん ) だが、日本( にっぽん ) 代表( だいひょう ) として試合( しあい ) に出( で ) る。
彼女( かのじょ ) は女性( じょせい ) としても女優( じょゆう ) としても最高( さいこう ) だが、妻( つま ) としてはどうだろう。
2. N／Aい／Aな／N（普通( ふつう ) 形( けい ) ）+　に/A（な）だ+に/Nだ+に/それにしてはしては
Cho dù là/ Dù là/ thế mà lại

Dùng khi muốn nói đến một sự việc khác biệt với lẽ thường. 

Dùng khi muốn nói lên cảm giác ngạc nhiên, bất ngờ, ngoài dự tính

Ví dụ
外国人( がいこくじん ) にしては日本語( にほんご ) が上手( じょうず ) だ。
初( はじ ) めてにしては、よくできました。
3. N／Aい／Aな／N（普通( ふつう ) 形( けい ) ）+　に/A（な）だ+に/Nだ+に/それにしても しても
Thậm chí là...cũng/ Dẫu là vậy thì...
Dùng để cường điệu hóa sự việc rằng dù ở mức độ thấp/cao hơn thì cũng không chấp nhận được
Ví dụ
これは、100円( えん ) にしても、高( たか ) いと思( おも ) う。
遅( おく ) れるとは聞( き ) いていたけど、それにしても田中( たなか ) さん、遅( おそ ) いね。
4. N／Aい／Aな／N（普通( ふつう ) 形( けい ) ）+　と　+　したら／すれば
Nếu là, nếu...thì
Sử dụng cho các câu điều kiện
Ví dụ
その話( はなし ) が本当( ほんとう ) だとしたら、うれしいです。
飛行機( ひこうき ) で行( い ) くとしたら、いくらぐらいかかりますか。
