1. 決( けっ ) して + Vない/ Aくない/ A（な）でない/ Nでない
Nhất quyết không, nhất định không, tuyệt đối không

Dùng khi muốn thể hiện quyết tâm không thực hiện một hành động nào đó

Ví dụ
私( わたし ) は決( けっ ) して夢( ゆめ ) をあきらめません。
「うそは決( けっ ) して申( もう ) しません。」と彼( かれ ) は言( い ) った。
2. まったく + Vない/ Aくない/ A（な）でない
Hoàn toàn không

Dùng khi khẳng định sự vật, sự việc hoàn toàn không có, không thể xảy ra.

Ví dụ
私( わたし ) はまったく泳( およ ) げない。
彼( かれ ) が怒( いか ) っている理由( りゆう ) は、私( わたし ) にはまったくわからない。
3. Nはめったにない/ めったにVない
Hiếm khi

Thường dùng trong trường hợp số lượng của sự vật, sự việc rất ít, hầu như hiếm khi có được, xảy ra

Ví dụ
こんなチャンスはめったにないよ。
忙( いそが ) しくてめったに休( やす ) みが取( と ) れない。
4. 少( すこ ) しも/ ちっとも + Vない/ Aくない/ A（な）でない
Một chút cũng không, hoàn toàn không

Dùng nhấn mạnh sự phủ định

Ví dụ
あの人( ひと ) が話( はな ) す英語( えいご ) は少( すこ ) しもわからない。
スタイルのことは、少( すこ ) しも気( き ) にならない。
