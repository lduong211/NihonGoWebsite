1. A。だけど、B。
A＞＜B
Nhưng mà....

Dùng khi muốn thể hiện ý ngược lại với câu trước (thường là mong muốn)

Ví dụ
旅行( りょこう ) に行( い ) きたい。だけどひまがない。
よくカラオケに行( い ) く。だけど歌( うた ) は下手( へた ) だ。
2. ＜理由( りゆう ) ／原因( げんいん ) ＞。/＜当然( とうぜん ) の結果( けっか ) ／判断( はんだん ) ＞。 + ですから（だから）
Vì vậy nên

Dùng cho câu kết quả, hệ quả của câu trước. Câu đi liền trước thường là câu nêu lý do, phán đoán, kết quả đương nhiên

Ví dụ
「天気( てんき ) 予報( よほう ) では午後( ごご ) から雨( あめ ) だそうです。ですから、傘( かさ ) を持( も ) っていったほうがいいですよ。」
明日( あした ) から旅行( りょこう ) に行( い ) きます。ですから、申( もう ) し訳( わけ ) ありませんが、来週( らいしゅう ) のパーテイーには出席( しゅっせき ) できません。
3. ＜予想( よそう ) ／予定( よてい ) ＞。/＜予想( よそう ) の結果( けっか ) ／事実( じじつ ) ＞。+ところが
Nhưng mà

Thường dùng trong trường hợp phủ định một suy đoán, dự định hoặc suy nghĩ về sự việc, hành động

Ví dụ
昨夜( さくや ) はコンサートに行( い ) くつもりだった。ところが病気( びょうき ) で行( い ) けなくなった。
田中( たなか ) さんは私( わたし ) より若( わか ) いと思( おも ) っていた。ところが私( わたし ) より５歳( とし ) も年上( としうえ ) だった。
4. A。ところで、B。
Nhân tiện thì, nhân dịp này thì

Dùng trong trường hợp muốn đề cập đến một vấn đề khác, đổi chủ đề giữa chừng câu chuyện

Ví dụ
「明日( あした ) 、試験( しけん ) でしょ。がんばってね。ところで、来週( らいしゅう ) の月曜日( げつようび ) は空( あ ) いてる？」
もうすぐ、今年( ことし ) も終( お ) わりですね。ところで、お正月( しょうがつ ) はどうなさいますか。
Danh sách tổng hợp
  • 1. ～行( い ) きたい。だけど、行( い ) けない。
  • 2.～雨( あめ ) です。ですから、傘( かさ ) がいります。
  • 3.～行( い ) く予定( よてい ) だった。ところが、行( い ) けなくなった
  • 4.～明日( あした ) は試験( しけん ) だ。ところで、来週( らいしゅう ) は？
