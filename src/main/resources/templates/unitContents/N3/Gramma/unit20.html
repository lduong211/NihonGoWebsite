1.V／Aだな／Nだの（普通( ふつう ) 形( けい ) ）おかげで
Nhờ có

Dùng để nhắc đến kết quả tốt đẹp nhờ vào sự việc, con người nào đó. (lý do)

Mang hàm ý biết ơn

Ví dụ
先生( せんせい ) のおかげで合格( ごうかく ) できました。
日本( にっぽん ) へ来( き ) たおかげで、日本語( にほんご ) が上手( じょうず ) になった。
2. V／A／N（普通( ふつう ) 形( けい ) ）せいで

Vì, do

せいか：Hình như là vì

Dùng khi muốn nhắc đến lý do liên đới tới một kết quả xấu

Mang hàm ý trách móc

Ví dụ
バスが遅( おく ) れたせいで、約束( やくそく ) の時間( じかん ) に間( ま ) に合( あ ) わなかった。
疲( つか ) れたせいか、頭( あたま ) が痛( いた ) い。
3. V／A／N（普通( ふつう ) 形( けい ) ） かわりに
Thay cho....

Dùng để thể hiện sự thay thế tương đương của sự vật, sự việc này với sự vật, sự việc khác

Ví dụ
車( くるま ) で来( き ) たので、ビールのかわりにジュースをください。
日曜日( にちようび ) に働( はたら ) いたかわりに、今日( きょう ) 休( やす ) みを取( と ) りました。
4. Nに かわって
Thay thế cho, thay mặt cho

Dùng để thể hiện sự thay thế tương đương của sự vật, sự việc, con người

Đi sau thường không dùng thể mệnh lệnh, sai khiến

Ví dụ
出張中( しゅっちょうちゅう ) の部長( ぶちょう ) にかわって、私( わたし ) がごあいさつさせていただきます。
野球( やきゅう ) にかわり、サッカーがさかんになってきた。
Danh sách tổng hợp
  • 1. ～先生( せんせい ) のおかげで
  • 2. ～遅( おく ) れたせいで
  • 3. ～ビールのかわりに
  • 4. ～部長( ぶちょう ) にかわって
