1.

Nという　+　N

Nという　+　もの

Nという　+　　こと

といった：Nêu lên ví dụ

Gọi là, tên là

Dùng khi nhắc đến sự vật, con người, sự việc mà bản thân mình/ đối phương không biết.

Dùng gọi tên hoặc định nghĩa

Ví dụ
さっき、木村( きむら ) さんという人( ひと ) から電話( でんわ ) がありましたよ。
これは、日本( にっぽん ) の楽器( がっき ) で「尺八( しゃくはち ) 」というものです。
2.

N　+　というのは

N　+　っていうのは

NというのはN’のことだ。

NといのうはN’という意味( いみ ) です。

Cái mà gọi là, cái mà tên là
Dùng khi muốn định nghĩa, giải thích về sự vật sự việc (với cụm từ muốn định nghĩa là chủ ngữ)
Ví dụ
「デジカメ」というのは、デジタルカメラを短( みじか ) くした言( い ) い方( かた ) です。
「電車( でんしゃ ) で「カクテイ」っていうのは何( なに ) のことですか。」
3.

文( ぶん ) （普通( ふつう ) 形( けい ) ）　+　というの

 

文( ぶん ) （普通( ふつう ) 形( けい ) ）　+　っていうの

 

文( ぶん ) （普通( ふつう ) 形( けい ) ）　+　ということ

 

文( ぶん ) （普通( ふつう ) 形( けい ) ）　+　っていうこと

Về việc...

Dùng khi muốn biến một câu văn thành cụm danh từ.

Dùng khi định nghĩa, giải thích hoặc nhắc đến một sự việc nào đó.

Ví dụ
リンさんが帰国( きこく ) するということを聞( き ) いて驚( おどろ ) きました。
こんなによく遅刻( ちこく ) をするというのは、問題( もんだい ) ですよ。
