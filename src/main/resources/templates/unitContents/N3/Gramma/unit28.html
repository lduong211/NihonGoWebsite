1. Vるまで
Cho đến khi (hành động)

Dùng cho trường hợp miêu tả điểm đến của một sự việc nào đó

Ví dụ
映画( えいが ) が始( はじ ) まるまで30分( ぶん ) あります。
連絡( れんらく ) があるまで待( ま ) っています。
2. Nまで（それまで、そこまで）
Cho đến mức là (danh từ), Đến cả

Dùng khi thể hiện sự ngạc nhiên về mức độ nhiều, quá của vấn đề

Ví dụ
この魚( さかな ) は骨( ほね ) まで食( た ) べられますよ。
あなたまで私( わたし ) を疑( うたが ) うのですか。
3. N１からN２にかけて

N: 時( とき ) 、場所( ばしょ )

Từ....(trải) đến....

Dùng trong trường hợp miêu tả khoảng thời gian/ không gian mà điểm bắt đầu và kết thúc không xác định chính xác

Ví dụ
明日( あした ) は昼( ひる ) から夕方( ゆうがた ) にかけて雨( あめ ) でしょう。
九州( きゅうしゅう ) から本州( ほんしゅう ) にかけて梅雨( つゆ ) 入( はい ) りました。
4. Nにおいて
Tại, ở

Dùng trong trường hợp nói đến thời gian hay địa điểm.

Là cách dùng trang trọng hơn so với "で"

Ví dụ
大阪( おおさか ) において、国際( こくさい ) 会議( かいぎ ) が行( おこな ) われた。
結果( けっか ) はホームページにおいて発表( はっぴょう ) されます。
Danh sách tổng hợp
  • 1. ～映画( えいが ) が始( はじ ) まるまで
  • 2.～骨( ほね ) まで食( た ) べられる
  • 3.～昼( ひる ) から夕方( ゆうがた ) にかけて
  • 4.～大阪( おおさか ) において
