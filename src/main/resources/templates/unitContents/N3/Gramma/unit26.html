1. Vます 上( あ ) げる/上( あ ) がる
Làm....xong

Dùng để nói đến việc hoàn thành hoàn toàn, kết thúc một công việc. Có thể dùng cả tự và tha động từ

Ví dụ
やっとレポートを書( か ) き上( あ ) げた。
ケーキが焼( や ) き上( あ ) がりましたよ。
2. Vます 切( き ) る/切( き ) れる
Vます切( き ) れない
(Động từ) hết.../ Không hết

Dùng khi sử dụng hết sự vật, sự việc nào đó, không còn sót lại

Ngược lại, với thể phụ định sẽ biểu hiện sự không kết thúc, kéo dài của hành động hoặc tính chất của sự vật

Ví dụ
ご飯( はん ) の量( りょう ) が多( おお ) くて、食( た ) べ切( き ) れないよ。
長( なが ) い小説( しょうせつ ) を、2日( ひ ) 間( かん ) で読( よ ) み切( き ) った。
3. Vます かける/かけのN/かけだ
Chưa xong, dở dang

Dùng trong trường hợp hành động đang còn được thực hiện (dù có thể không liên tục) , vẫn chưa kết thúc hoàn toàn

Ví dụ
この本( ほん ) はまだ読( よ ) みかけだ。
おふろに入( はい ) りかけたときに電話( でんわ ) が鳴( な ) った。
4. VますたてのN
Vますたてだ

Không dùng: たてる、読( よ ) みたて、寝( ね ) たて、食( た ) べたて

Mới...xong

Dùng trong trường hợp hành động vừa mới kết thúc

Thường dùng với nghĩa "tươi, mới, nóng hổi..."

Ví dụ
焼( や ) きたてのパンはおいしい。
あのスーパーは、とりたての新鮮( しんせん ) な野菜( やさい ) を売( う ) っている。
Danh sách tổng hợp

  • 2.～食( た ) べない
  • 3.～読( よ ) みかけの本( ほん )
  • 4.～焼( や ) きたてのパン
