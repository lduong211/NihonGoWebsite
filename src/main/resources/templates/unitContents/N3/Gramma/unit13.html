1.

Vても

 

Aくても

 

Aな／Nでも

Dù là, cho dẫu là
Dùng để nói đến sự việctrở nên ngược lại với điều đáng lẽ diễn ra.
Ví dụ
調( しら ) べてもわからなかったので、先生( せんせい ) に聞( き ) いた。
必要( ひつよう ) なので、高( たか ) くても買( か ) います。
2. どんなに/いくら Vても/Aくても/A（な）でも/Nでも
Cho dẫu có như thế nào/ Cho dẫu là bao nhiêu đi nữa
Dùng khi muốn cường điệu hóa sự việc, nhấn mạnh sự việc không thể/có thể ở mức độ cao
Ví dụ
ギターを習( なら ) い始( はじ ) めたが、どんなに練習( れんしゅう ) しても全然( ぜんぜん ) うまくならない。
私( わたし ) は、いくらお酒( さけ ) を飲( の ) んでも顔色( かおいろ ) が変( か ) わらない。
3. Vない +　ずに
しない → せずに
Không làm gì (mà làm)
Dùng trong trường hợp chúng ta bỏ qua 1 hành động mà tự nhiên, bình thường sẽ làm
Ví dụ
辞書( じしょ ) を使( つか ) わずに書( か ) いたので、自信( じしん ) がありません。
昨夜( さくや ) 、歯( は ) をみがかずに寝( ね ) てしまった。
Danh sách tổng hợp
