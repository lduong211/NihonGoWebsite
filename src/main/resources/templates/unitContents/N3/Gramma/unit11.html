1. N／Aな／V／Aい（普通( ふつう ) 形( けい ) ）と　+　いうより／いうか
Thay vì nói là....( thì đúng hơn là....)

Dùng khi muốn so sánh mang tính chất cường điệu hơn bình thường. (Mức độ thấp/ cao hơn)

Thường dùng trong câu cảm thán mang nghĩa tiêu cực

Ví dụ
前( まえ ) の車( くるま ) は遅( おそ ) すぎて、走( はし ) るというよりはっているようだ。
今日( きょう ) は涼( すず ) しいというより寒( さむ ) いくらいだった。
2. N／Aな／V／Aい（普通( ふつう ) 形( けい ) ）と いうと/いえば/いったら
Khi nhắc đến/ Nếu nói đến
Dùng khi muốn nói đến một điểm nổi bật nào đó của sự vật, sự việc
Ví dụ
京都( きょうと ) というと、お寺( てら ) をイメージします。
日本( にっぽん ) の食( た ) べ物( もの ) といえば、おすしがいちばん有名( ゆうめい ) だと思( おも ) います。
3. N／Aな／V／Aい（普通( ふつう ) 形( けい ) ）と　いっても
Dẫu nói là....

Dùng khi muốn đính chính một vấn đề, sự việc có mức độ thấp hơn so với tên gọi của nó

Hoặc dùng khi muốn so sánh mức độ cao thấp với 1 sự việc khác

Ví dụ
週末( しゅうまつ ) は旅行( りょこう ) しました。旅行( りょこう ) といっても、近( ちか ) くの温泉( おんせん ) に行( い ) っただけですが。
今週( こんしゅう ) は忙( いそが ) しいといっても来週( らいしゅう ) ほどじゃない。
Danh sách tổng hợp
  • 1. ～走( はし ) るというより
  • 2. ～京都( きょうと ) というと
  • 3.～旅行( りょこう ) といっても
