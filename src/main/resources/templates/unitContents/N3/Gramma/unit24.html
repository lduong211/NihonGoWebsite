1. つまり
Tức là

Dùng để lặp lại một sự vật, sự việc với một cách nói, cách gọi khác tương đương

Ví dụ
父( ちち ) の兄( あに ) 、つまり私( わたし ) の伯父( おじ ) は、医者( いしゃ ) をしている。
田中( たなか ) さんは、携帯( けいたい ) もパソコンも持( も ) っていない。つまり、メールで連絡( れんらく ) はできないのだ。
2. ＜目的( もくてき ) ・理由( りゆう ) ＞。そのため（に）＜結果( けっか ) ＞。
Vì thế, Do đó

Dùng làm từ nối tiếp diễn với lý do, mục tiêu được nêu ra trước đó để tiếp nối đến kết quả

Ví dụ
隣( となり ) の駅( えき ) で事故( じこ ) があったらしい。そのために電車( でんしゃ ) が遅( おく ) れている。
留学( りゅうがく ) するつもりだ。そのために、バイトしてお金( かね ) をためている。
3. ＜過去( かこ ) に起( お ) きたこと＞
その結果( けっか ) ＜が原因( げんいん ) ／理由( りゆう ) で起( お ) きたこと＞
Kết quả là

Dùng trong trường hợp một sự việc đã xảy ra trong quá khứ là lý do dẫn đến hậu quả của sự việc tiếp theo. Dùng làm từ nối 2 câu.

Ví dụ
父( ちち ) は、人( ひと ) の何倍( なんばい ) も努力( どりょく ) した。その結果( けっか ) 、仕事( しごと ) で成功( せいこう ) した。
3ヶ月( かげつ ) ダイエットを続( つづ ) けた。その結果( けっか ) 、5キロやせた。
4. ＜結論( けつろん ) ＞ なぜなら（ば）/なぜかというと/どうしてかというと ＜原因( げんいん ) ＞/ ＜理由( りゆう ) ＞
Nếu nói tại sao thì là vì..., Nếu hỏi sao thì là do,..., Vì
Dùng để nêu lý do, nguyên nhân của một kết luận được nêu trước đó
Ví dụ
来週( らいしゅう ) 、国( くに ) に帰( かえ ) る予定( よてい ) です。なぜなら（ば）、親友( しんゆう ) の結婚式( けっこんしき ) に出席( しゅっせき ) するからです。
学校( がっこう ) を変( か ) えた。なぜかというと、ぼくのレベルのクラスがなかったからだ。
Danh sách tổng hợp
  • 1. ～父( ちち ) の兄( あに ) 、つまり伯父( おじ )
  • 2.～事故( じこ ) があった。そのため（に）遅( おく ) れている
  • 3.～努力( どりょく ) した。その結果( けっか ) 成功( せいこう ) した
  • 4.～帰国( きこく ) する。なぜなら結婚式( けっこんしき ) に出席( しゅっせき ) するからだ。
