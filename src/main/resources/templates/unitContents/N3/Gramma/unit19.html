1.Nに とって/とっては/とっても
Đối với

Dùng để thể hiện cách nhìn về sự việc, sự vật từ vị trí, quan điểm của cá nhân, hoặc dùng nói đến các sự việc có liên quan

Không dùng được với các danh từ đứng trước là chủ thể thực  hiện hành động chính trong câu.

Ví dụ: この仕事( しごと ) はわたしにとってできない。

Ví dụ
あなたにとって、いちばん大切( たいせつ ) なものは何( なん ) ですか。
このあたりの人( ひと ) 々にとっては、車( くるま ) は必需品( ひつじゅひん ) です。
2. V／Aだな／Nだの（普通( ふつう ) 形( けい ) ） わりに
Cho dẫu, mặc cho

Dùng khi miêu tả, thể hiện trường hợp sự vật, sự việc trái ngược với lẽ thường, lẽ tự nhiên.

Thể hiện cảm xúc ngạc nhiên, bất ngờ, ngoài dự tính

Ví dụ
年( とし ) のわりには若( わか ) く見( み ) える。
お金( かね ) がない（と言( い ) っている）わりに、よく買( か ) い物( もの ) をするね。
3. V／ Aだな／Nだの （普通( ふつう ) 形( けい ) ）くせに
Thế mà

Dùng trong trường hợp muốn phàn nàn, khó chịu về sự việc không hay đã xảy ra trong khi đáng lẽ không phải như vậy.

Người dùng mang cảm giác không vui, khó chịu về vấn đề

Ví dụ
知( し ) っているくせに、教( おし ) えてくれない。
元気( げんき ) なくせに、病気( びょうき ) のふりをしている。
4. V／A／N（普通( ふつう ) 形( けい ) ）／文( ぶん ) －　なんて／など
Mấy cái thứ như là, mấy chuyện kiểu như

Dùng trong trường hợp muốn đánh giá thấp một sự việc, sự vật nào đó.

Người nói muốn nhấn mạnh sự phủ định với sự việc sự vật. Theo sau thường là động từ mang ý phủ định

Ví dụ
お化粧( けしょう ) なんかしてはいけません。
納豆( なっとう ) なんて嫌( きら ) いだ。
Danh sách tổng hợp
  • 1. ～あなたにとって
  • 2. ～年( とし ) のわりには
  • 3. ～知( し ) っているくせに
  • 4. ～お化粧( けしょう ) なんか
