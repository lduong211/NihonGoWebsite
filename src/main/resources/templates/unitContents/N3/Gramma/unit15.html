1.Vる／Vない　+　つもりだった
Đã dự định làm gì/ không làm gì
Dùng để nói đến một dự định, ý định thực hiện một hành động nào đó trong quá khứ nhưng sau đó lại không thực hiện như vậy
Ví dụ
昨日( きのう ) は買( か ) い物( もの ) に行( い ) きつもりでしたが、頭( あたま ) が痛( いた ) かったのでずっと家( いえ ) にいました。
お菓子( かし ) を食( た ) べないつもりでしたが、つい食( た ) べてしまいました。
2. V／A／N（普通( ふつう ) 形( けい ) ）/Aだな/Nだの はずだ/はずがない
Chắn chắn là/ chắn chắn không

Dùng thể hiện ý khẳng định của người nói dựa trên một căn cứ nào đó.

Riêng cụm "はずがない" mang ý nghĩa phủ định mạnh hơn.

Ví dụ
田中( たなか ) さんは今( いま ) 、旅行中( りょこうちゅう ) だから、家( いえ ) にいないはずだ。
まじめな田中( たなか ) さんが、無断( むだん ) で休( やす ) むはずがない。
3. Vる/A（な）である/A（い）くある/べきだ/べきではない

（すべき　＝　するべきだ）

Phải, nên làm gì/ không làm gì

Dùng để thể hiện việc ở một tình huống nào đó thì phải (mang ý bắt buộc) thực hiện /không thực hành động nào đó. Nếu không làm được thì không thể chấp nhận được ở vị trí đó, tình huống đó.

Ví dụ
約束( やくそく ) は守( まも ) るべきだ。
おもちゃは、まず安全( あんぜん ) であるべきだ。
4. Vた/Vなかったものだ/もんだ
Đã thường làm gì/ không làm gì

Dùng để nhắc đến những việc trong quá khứ. Thường là thói quen, những hành động thường xuyên.

Nhấn mạnh về kỷ niệm, hồi ức

Ví dụ
子( こ ) どものころはよく川( かわ ) で遊( あそ ) んだものだ
学生( がくせい ) 時代( じだい ) は毎日( まいにち ) 図書館( としょかん ) へ通( とお ) ったものだ。
Danh sách tổng hợp
  • 1. ～行( い ) くつもりでした
  • 2. ～家( いえ ) にいないはずだ
  • 3.～守( まも ) るべきだ
  • 4. ～よく遊( あそ ) んだものだ
