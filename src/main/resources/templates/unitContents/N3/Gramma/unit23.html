1. Nだ／Nだったっけ
A（な）だ／Aだったっけ
Aかったっけ
Vたっけ
でしたっけ／ましたって
Phải vậy không?

Dùng để xác nhận lại lần nữa sự việc, sự vật, đặc điểm

Thường dùng trong thể nói thông dụng, thân mật

Ví dụ
国( くに ) へ帰( かえ ) るのは、来週( らいしゅう ) だ（った）っけ？
明日( あした ) のパーテイーのこと、話( はな ) し（まし）たっけ？
2. Vるしか　ない
Chỉ còn cách, chẳng còn cách nào khác ngoài

Dùng khi muốn nhấn mạnh vào tình huống khó khăn và chỉ còn một cách giải quyết

Ví dụ
できるまで、やるしかない。
直( なお ) せないから、新( あたら ) しいのを買( か ) うしなかった。
3. V／A／N（普通( ふつう ) 形( けい ) ）（んだ）って
N／Aな（なん）だって
～までだて／～からだって
～だけだって
Nghe nói là

Dùng khi muốn tường thuật lại sự việc nghe được

Thường dùng trong thể văn nói thân mật, thông dụng.

Nữ thường dùng "でして"

Ví dụ
「田中( たなか ) さん、結婚( けっこん ) してるんだって。」「へー、知( し ) らなかった。」
「試験( しけん ) の範囲( はんい ) は、教科書( きょうかしょ ) の最初( さいしょ ) から50ページまでだって。」
4. V／A／N（普通( ふつう ) 形( けい ) ）（んだ）もん
N／Aな（なん）だもん
Thì là vì....

Dùng để giải thích lý do kèm theo tâm trạng không vui vẻ, khó chịu, thất vọng

Thường dùng trong văn nói thân mật, thông dụng

Ví dụ
「どうして食( た ) べないの？」「だって、まずいんだもん。」
今日( きょう ) の試験( しけん ) 、できなかった・・・。勉強( べんきょう ) しなかったんだもん、仕方( しかた ) がない。
