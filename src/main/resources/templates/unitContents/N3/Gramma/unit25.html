1. NはもちろんN’も
［文( ぶん ) ］のはもちろん～（も）
(Danh từ) là đương nhiên rồi nhưng cả (Danh từ ')

Dùng để thể hiện một sự vật sự việc cũng có một đặc điểm (tốt) tương tự như một đặc điểm đương nhiên khác.

Thường thể hiện ý khen ngợi, hay ngạc nhiên

Ví dụ
彼( かれ ) は勉強( べんきょう ) はもちろんスポーツもよくできる。
キャベツは炒( いた ) めて食( た ) べるはもちろん、生( なま ) で食( た ) べてもおいしいです。
2. V／Aだな／Nだ（普通( ふつう ) 形( けい ) ） ばかりか・・・（も）/ばかりでなく・・・（も）
Không chỉ có....mà còn...

Dùng khi thể hiện sự việc, sự vật có thêm một đặc điểm (không tốt) ngoài đặc điểm đã biết

Thể hiện sự khó chịu, thất vọng, không hài lòng

Ví dụ
あの店( みせ ) は、味( あじ ) が悪( わる ) いばかりか、店員( てんいん ) の態度( たいど ) もひどい。
私( わたし ) は、漢字( かんじ ) ばかりか、まだひらがなも書( か ) けません。
3. Nに比( くら ) べて
［文( ぶん ) ］のに比( くら ) べ
So với, so sánh với

Dùng trong trường hợp so sánh giữa hai sự vật, sự việc

Ví dụ
試験( しけん ) 問題( もんだい ) は昨年( さくねん ) に比( くら ) べて易( やさ ) しくなった。
店( みせ ) で買( か ) うのに比( くら ) べ、通信販売( つうしんはんばい ) は便利( べんり ) だが、欠点( けってん ) もある。
4. Nに/[文 ( ぶん ) ]のに 対( たい ) して/対( たい ) し/対( たい ) しては/対( たい ) しても/対( たい ) するN
Đối với, đối với...thì, đối với...cũng, ....đối với....

Dùng thể hiện thái độ, hành động đối với một sự việc vấn đề

Hoặc dùng khi thể hiện hai sự việc, sự vật đối lập nhau

Ví dụ
田中( たなか ) 先生( せんせい ) は、生徒( せいと ) に対( たい ) して厳( きび ) しい。
まじめな長男( ちょうなん ) に対( たい ) して、次男( じなん ) は遊( あそ ) んでばかりで学校( がっこう ) もよく休( やす ) む。
