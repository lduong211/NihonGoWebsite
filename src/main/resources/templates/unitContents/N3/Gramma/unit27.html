1. V／A／N（普通( ふつう ) 形( けい ) ）と　いい（のに）なあ。
Vたら／Aたら/A（な）たら／Nたら/Vば／Aければ/Aなら（ば）/Nなら（ば） （いいのに）なあ
Ước gì..., mong sao

Dùng để thể hiện ý muốn, mong ước của cá nhân. Thuộc dạng câu cảm thán

Ví dụ
もっと日本語( にほんご ) がうまく話( はな ) せたらいいなあ。
明日( あした ) 、雨( あめ ) が降( ふ ) らないといいなあ。
2. Vば/Vなければ/Vたら/Vなかったら よかった
Giá mà .... thì đã tốt rồi

Dùng thể hiện ý hối hận về sự việc đã xảy ra không như mong muốn của bản thân

Ví dụ
遅刻( ちこく ) してしまった。もっと早( はや ) く家( いえ ) を出( で ) ればよかった。
田中( たなか ) さんにあんなことを言( い ) わなければよかった。
3. Vば～/Vたら～のに
Thế mà.../ Giá mà...

Dùng thể hiện ý tiếc nuối về sự việc đã xảy ra không như mong muốn của bản thân

Ví dụ
パーテイー、楽( たの ) しかったよ。君( きみ ) も行( い ) けばよかったのに。
安( やす ) かったら買( か ) うのに。
4. Vる/Vない かな（あ）
Hay sao.., mong là..., có...không đây

Dùng trong trường hợp thể hiện sự hi vọng, hay sự nghi vấn của bản thân về một vấn đề nào đó

Đa phần có hàm ý sự lo lắng, băn khoăn về sự việc xảy ra

Ví dụ
バス、早( はや ) く来( こ ) ないかなあ。
この実験( じっけん ) 、うまくいくかな。
Danh sách tổng hợp
  • 1. ～うまく話( はな ) せるといいなあ
  • 2.～もっと早( はや ) く家( いえ ) をばよかった
  • 3.～あなたも行( い ) けばよかったのに
  • 4.～早( はや ) く来( こ ) ないかなあ。
