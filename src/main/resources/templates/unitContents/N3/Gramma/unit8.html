1. Nに／～のに　+ 関( かん ) して・関( かん ) しては・関( かん ) しても・関( かん ) するN
Liên quan đến
Dùng khi nói đến một phần tính chất có liên quan đến vấn đề
Ví dụ
この問題( もんだい ) に関( かん ) して、ご意見( いけん ) ありませんか。
農業( のうぎょう ) に関( かん ) する論文( ろんぶん ) を読( よ ) む。
2. Nに　+　ついて／ついての／ついては／ついても
Về....

Dùng khi nói về một hành động, sự việc tập trung vào vấn đề nào đó

Ví dụ
日本( にっぽん ) 文化( ぶんか ) について勉強( べんきょう ) する。
「受験( じゅけん ) についての注意( ちゅうい ) 」を読( よ ) む。
3. Nに　+　よれば／よると

Theo...

Dùng khi nói đến một sự việc có nguồn xuất phát từ những gì bản thân nghe được, nhìn thấy,...

Ví dụ
天気( てんき ) 予報( よほう ) によれば、明日( あした ) は晴( は ) れるらしい。
さっきの地震( じしん ) はテレビの速報( そくほう ) によると震度( しんど ) ３だそうだ。
4. Nに　+　よって／よっては／より／よるN

Bằng cách, do, vì, tùy theo

Dùng để thể hiện phương pháp, nguyên nhân, trường hợp cho từng hành động, sự việc nào đó

Ví dụ
彼( かれ ) は努力( どりょく ) によって病気( びょうき ) を克服( こくふく ) した！
人( ひと ) によって考( かんが ) え方( かた ) が違( ちが ) います。
