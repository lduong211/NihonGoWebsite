1.Nの／Vる／Vたついでに
Nhân tiện (làm gì) thì .....
Một hành động chính nào đó đang thực hiện thì nhân tiện đó thực hiện thêm một hành động phụ khác
Ví dụ
散歩( さんぽ ) のついでに、この手紙( てがみ ) を出( だ ) してきてくれませんか。
郵便局( ゆうびんきょく ) へ行( い ) ったついでに、はがきを買( か ) ってきた。
2. Nの／Vるたびに
Mỗi khi, mỗi lần

Dùng trong trường hợp nhắc đến một sự việc thường xảy ra mỗi khi có 1 tình huống cụ thể nào đó

Ví dụ
買( か ) い物( もの ) のたびに、袋( ふくろ ) をたくさんもらう。
この曲( きょく ) を聞( き ) くたびに、ふるさとを思( おも ) い出( だ ) す。
3. Vた。 とたん
Vừa mới (làm gì) thì....

Dùng trong trường hợp một sự việc xảy ra ngay khi một hành động nào đó vừa được thực hiện.

Vế sau không dùng các mẫu câu nghĩa chủ động (tha động từ) hoặc dùng thể sai khiến, bắt buộc

Ví dụ
窓( まど ) を開( あ ) けたとたん、強( つよ ) い風( かぜ ) が入( はい ) ってきた。
お酒( さけ ) を飲( の ) んだとたん、顔( かお ) が赤( あか ) くなった。
4. Nの／Vている最中( さいちゅう )
Trong khi, đang lúc, giữa lúc

Dùng nói đến một hành động đột ngột diễn ra giữa tiến trình đang diễn ra của một hành động khác

Ví dụ
食事( しょくじ ) の最中( さいちゅう ) に、お客( きゃく ) さんが来( き ) た
会議( かいぎ ) をしている最中( さいちゅう ) に、携帯( けいたい ) 電話( でんわ ) が鳴( な ) った。
Danh sách tổng hợp

  • 3.～開( あ ) けたとたん
  • 4. ～食事( しょくじ ) の最中( さいちゅう ) に
