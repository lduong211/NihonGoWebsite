1. Vようと思( おも ) う
Dự định làm gì
Dùng để thể hiện ý chí, dự định của chủ thể
Ví dụ
会社( かいしゃ ) をやめようと思( おも ) っている。
来年( らいねん ) 、国( くに ) に帰( かえ ) ろうと思( おも ) っています。
2.

Vようとする

(vừa ) Định làm gì

Dùng để diễn tả ý định của chủ thể trong khoảnh khắc, khoảng thời gian ngắn. Dùng với những hành động mang tính ngắn hạn, thường ngày.

Ví dụ
電車( でんしゃ ) に乗( の ) ろうとしたときに、ドアが閉( し ) まって乗( の ) れなかった。
あ、ポチが、あなたの靴( くつ ) をかもうとしているよ。
3. Vようとしない

Dùng để thể hiện việc không cảm nhận được ý chí thực hiện hành động hay động tác

Thường dùng để thể hiện hi vọng, cầu mong về một việc gì đó
Ví dụ
「ポチ、どうしたんだろう。ご飯( はん ) を食( た ) べようとしないんだよ。」
彼( かれ ) は自分( じぶん ) のことは何( なに ) も言( い ) おうとしない。
