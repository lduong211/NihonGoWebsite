1. もし + Vた（な）ら/Aかった（な）ら/N／A（な）だった（な）ら
Giả sử là...thì....

Dùng khi muốn đưa ra giả thuyết về một sự việc không có thật và hệ quả của sự việc không có thật đó

Ví dụ
もし試験( しけん ) を受( う ) けていたなら、合格( ごうかく ) していたと思( おも ) う。
もし彼( かれ ) が社長( しゃちょう ) でなかったなら、会社( かいしゃ ) はつぶれていたと思( おも ) う。
2. もしV／A／N（普通( ふつう ) 形( けい ) ）+と　しても/と　したって
Giả sử là....thì cũng

Dùng khi nói đến giả định mang tính khả thi thấp

Ví dụ
もし休( やす ) みが取( と ) れたとしても、旅行( りょこう ) に行( い ) かないつもりです。
もし、お金( かね ) がたくさんあったとしても、そんなものは買( か ) わない。
3. もしも/ もしもの場合( ばあい ) / もしもの時( とき ) +V／A／N（普通( ふつう ) 形( けい ) ）なら/N／（Aな）なら/V／Aても/N／（Aな）でも V／A／Nたら
Giả sử, nếu....thì/ thì cũng

Dùng cho trường hợp đưa ra điều kiện giả định

Ví dụ
もしも生( う ) まれ変( か ) われるなら、男( おとこ ) になりたい。
もしも地震( じしん ) が起( お ) きても、この家( いえ ) 、丈夫( じょうぶ ) だから倒( たお ) れないでしょう。
Danh sách tổng hợp
  • 1.～もし受験( じゅけん ) していたなら
  • 2.～もし休( やす ) みがたとしても
  • 3.～もしも生( う ) まれ変( か ) われるなら
