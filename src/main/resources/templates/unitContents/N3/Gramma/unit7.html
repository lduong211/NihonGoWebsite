1.

N／Vて　+　ばかり／ばかりだ／ばかりのN／ばかりで

Vてばかりいる～

Toàn là, đầy...

Một hành động nào đó được thực hiện liên tục

Sự vật, chủ thể nào đó hiện hữu với số lượng nhiều

Ví dụ
この店( みせ ) のお客( きゃく ) さんは、女性( じょせい ) ばかりですね。
弟( おとうと ) は、テレビを見( み ) てばかりいる。
2. NだけしかVない
Chỉ có

Dùng thể hiện ý duy nhất, độc nhất

Ví dụ
このコンサートは、会員( かいいん ) だけしか入( い ) れません。
今日( きょう ) はお客様( きゃくさま ) が一人( ひとり ) だけしか来( き ) ませんでした。
3. N／Nに／Nで　+　さえ

Thậm chí là, đến cả

Dùng để thể hiện một việc ở mức độ đặc biệt cao/thấp mà chắc chắn có thể / không thể làm được.
Ví dụ
ひらがなさえ書( か ) けないんですから、漢字( かんじ ) なんて書( か ) けません。
そんなこと、子( こ ) どもでさえ知( し ) っている。
4. N／～から（理由( りゆう ) ）　+　こそ

Chính là,. chính vì (lý do)

Dùng để nhấn mạnh khi chủ thể là tác nhân chính, lý do chính, cường điệu hóa lên

Ví dụ
明日( あした ) こそ勉強( べんきょう ) するぞ！
愛情( あいじょう ) があるからこそ、しかるんです。
