1. V（普通( ふつう ) 形( けい ) ）／Nの　+　ように
Như là ...., giống như là

Thường dùng để làm lời mở đầu trước khi giải thích về vấn đề gì đó

Ngoài ra còn dùng trong việc hướng dẫn người khác làm gì

Ví dụ
皆様( みなさま ) ご存( ぞん ) じのように、試験( しけん ) の内容( ないよう ) が変( か ) わります。
ここに書( か ) いてあるように、今度( こんど ) の木曜日( もくようび ) 、授業( じゅぎょう ) は午前中( ごぜんちゅう ) だけです。
2.

Vる／Vない +　ように

Hãy...làm gì/ Không được làm gì
Dùng để yêu cầu, chỉ định người khác (bậc dưới) làm gì
Ví dụ
ここでたばこを吸( す ) わないように。
ここでたばこを吸( す ) わないように。
「わかりました。」
3.

Vます／Vません／Vれます +　ように

Để, mong, cầu cho
Thường dùng để thể hiện hi vọng, cầu mong về một việc gì đó
Ví dụ
合格( ごうかく ) しますように。
母( はは ) の病気( びょうき ) が治( なお ) りますように。
Danh sách tổng hợp
  • 1. ～ご存( ぞん ) じのように
  • 2. ～早( はや ) くように
  • 3. ～合格( ごうかく ) しますように
