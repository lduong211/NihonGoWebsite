1.

Aい +　さ

 

A（な）だ +　さ

 

よい → よさ

例( れい ) ：うれしさ、暑( あつ ) さ、甘( あま ) さ、まじめさ

(Danh từ)

Dùng khi biến đổi tính từ thành danh từ

Thường dùng với những từ miêu tả tính chất, cảm xúc

Ví dụ
大( おお ) きさは違( ちが ) うが、君( きみ ) と同( おな ) じかばんを持( も ) っているよ。
子( こ ) どもに命( いのち ) の大切( たいせつ ) さを教( おし ) えなければならない。
2.

Aい　+　み

 

A（な）だ　+　み

例( れい ) ：悲( かな ) しみ、弱( よわ ) み、痛( いた ) み、真剣( しんけん ) み

ダメ：大( おお ) きみ　うれしみ 暑( あつ ) み　まじめみ

(Danh từ)

Dùng khi biến đổi tính từ thành danh từ.

Thường dùng với từ miêu tả trạng thái, tính chất tự nhiên

Ví dụ
戦争( せんそう ) が終( お ) わった今( いま ) でも、この国( くに ) の苦( くる ) しみはまだ続( つづ ) いている。
田中( たなか ) さんの強( つよ ) みは、2ヶ国( かこく ) 語( ご ) が話( はな ) せるということです。
3.

Nの　+　こと

 

V／A（い）／Nに　+　こと

 

Aだな／Nだである　+　こと

Về việc...

Dùng khi nói đến một chủ đề có liên quan đến hành động, sự việc nào đó

Thường dùng với ý nghĩa lịch sự/ trang trọng

Ví dụ
来週( らいしゅう ) のテストのことで、質問( しつもん ) があります。
田中( たなか ) さんが入院( にゅういん ) したことを知( し ) っていますか。
4.

V／Aい／N　+　の

 

Nだな／Aだな　+　の

例( れい ) ：見( み ) ているのが

いいのを選( えら ) んでください

暑( あつ ) いのはきらいだ

元気( げんき ) なのがいちばんだ

Việc (danh từ)

Dùng để biến đổi danh từ, động từ, tính từ, cụm câu thành cụm danh từ

Dùng trong trường hợp thân mật, ngắn gọn

Ví dụ
田中( たなか ) さんに電話( でんわ ) するのをすっかり忘( わす ) れていました。
日曜日( にちようび ) に混( こ ) んだところへ行( い ) くのは、あまり好( す ) きではありません。
