1.N／Aな／Aい／Vる くらい／ぐらい／ほど
Đến cỡ, đến mức, cỡ
Dùng để nói đến mức độ, quy mô dùng so sánh với sự vật, sự việc
Ví dụ
それは米粒( こめつぶ ) くらいの大( おお ) きさです。
この洗剤( せんざい ) は、おもしろいほど汚( よご ) れが落( お ) ちる。
2. Aい/Aなほど
Phía, bên...(hơn)

Dùng để nói đến một phía, một bên nào đó thì sẽ có đặc điểm hơn/kém hơn như thế nào

Ví dụ
若( わか ) い人( ひと ) ほど朝寝坊( あさねぼう ) をする。
伝統的( でんとうてき ) なものは、古( ふる ) いほど価値( かち ) がある。
3. VばVるほど
AければAいほど
AならばAなほど
Càng....càng....

Dùng để thể hiện mức độ kết quả càng tăng khi đặc tính của sự vật càng tăng

Ví dụ
彼( かれ ) のことを知( し ) れば知( し ) るほど好( す ) きになる。
荷物( にもつ ) は少( すく ) なければ少( すく ) ないほどいい。
4. NくらいN［人( ひと ) ］はいない
NほどN［物( もの ) ・事( こと ) ]は　ない
Cỡ như....thì không....

Dùng trong trường hợp đề cao mức độ của sự vật, sự việc, con người.

Mang ý khẳng định ngoài sự vật, sự việc, con người đó thì không có tương đương để thay thế

Ví dụ
彼女( かのじょ ) くらい親切( しんせつ ) な人( ひと ) はいない。
今年( ことし ) ほど雪( ゆき ) の降( ふ ) った年( とし ) はなかった。

  • 3. ～知( し ) ればほど
